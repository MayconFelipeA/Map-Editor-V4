# Map-Editor-V4

Original [fusez Map-Editor-V3](https://github.com/fusez/Map-Editor-V3)

## What's New?
**v4.0**
* **Required Pawn.CMD**
* **Added text3d in the object for editing: /edit.**
* **Added commands to show and hide text3d: /show.**
* **Added an option to set a new object model.**
